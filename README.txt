u-tech-beginners

u-Techのプログラミング初心者勉強会に参加してくださってありがとうございます。
最初に幾つかu-Techから重要な連絡をさせていただきます。

まず、本ゼミは特殊なものです。
u-Techのゼミは本来参加者全員が主体的に学んでいくものであるのに対し、本ゼミはアドバイザーのような形で代表の渡邉が参加しております。
原則的にこのような形でアドバイザーがいるのは本ゼミのみなのでご注意ください。

また、みなさんにお願いがあります。
本ゼミに参加するみなさんには1年以内にu-Techの何らかの仕事を手伝っていただきたいです。
えっ。って思った人もいると思いますが、説明させてください。

当たり前の話ですが、サークルの活動に参加することは多くのメリットがあります。(今参加しているみなさんのように)
しかし、サークルに活動を支えるのもそこそこ大変なわけです。(新歓活動、ビラ製作、コンパなどなど)
また、u-Techのゼミに参加されてる方はほんの少しでも構わないので、仕事をしていただくことになっています。

そこで、こちらからこのようなゼミを提供する代わりに、みなさんにも何らかの仕事を手伝っていただきたいのです。

仕事といっても、テント列のシフトに1時間30分だけ入るとか、ビラ詰めをやるとか、連絡委員会に代理で出席するとか、半日もかからないようなことを1度やってもらうだけで構いません。(これだけで1年間ゼミに出席し放題ですよ！)

このような仕事でしたらはるかにメリットの方が大きいと思うのですが、どうでしょうか?

納得していただけましたら、早速内容に入っていきたいと思います。

今回の初心者勉強の目的は「プログラマーとして自立すること」です。
ここにいるみなさんは自学すらままならない方が多いと思います。
それは仕方のないことで、プログラミングは始めたばかりの頃が一番たいへんで、慣れてくると急速に学習速度が上がっていきます。

ですので、その初歩を僕がサポートをしつつ、みなさんで協力して学習していっていただきたいと考えています。

また、教材はhttp://diveintopython3-ja.rdy.jp/index.htmlを用います。

最初に少しだけなぜPythonを学ぶのかについて説明します。

大きな利点はまず2つあります。

1つは、モダンな言語であり、一般的に評価が高い言語であるということ。

2つ目は、用途が多いことです。

さて、初日はおそらくPython3を使用するための準備で終わってしまうと思うので、その準備をこれからします。
